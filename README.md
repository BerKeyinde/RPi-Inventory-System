# RPi-Inventory-System
RPi based Inventory System with RFID Reader, RaspiCam and SMS Notification

Components used:
- Raspberry Pi Model B+
- Raspberry Pi Camera Module
- HY 8861 RFID Reader
- Huawei E353 USB 3G Dongle
- TP-Link TL-WN725N
 
Programs used:
- Apache2
- PHP5
- PHP5liteadmin
- RPi.GPIO
- Gammu
- SQLite3
